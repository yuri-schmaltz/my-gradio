# Instrumentação e validação mínima Gradio

## Logs
- logging estruturado backend (INFO, WARNING, ERROR)
- logs de inicialização e falha

## Métricas
- tempo de build/teste
- cobertura de testes
- badge Codecov

## Checklist de estados UI
- loading, erro, sucesso, vazio

## Validação de dependências
- script para checar versões e alertar desatualizações

## Checklist de acessibilidade mínima
- labels, foco, contraste (quando aplicável)

## Se faltar dados
- coletar logs de execução
- coletar métricas de build/teste
- garantir badge de cobertura
